Open source macOS tool

NoShot

A dirt-simple screenshot helper that saves, copies, and can ask local Codex to explain what you captured.

View source colinknapp.com

What it does

Fast capture

Command+Shift+1 captures a region, +2 captures a window, and +3 captures the full screen.

Save and copy

Images save to ~/Pictures/Greenshot by default and copy to the clipboard immediately.

Codex prompts

Command+Shift+4 asks Codex to explain a screenshot. +5 captures first, then asks for your custom prompt.

Build from source

git clone https://github.com/Leopere/noshot.git
cd noshot
make verify
make run

NoShot currently targets macOS and uses Apple’s built-in screencapture command.
